# Installation

## Stable release

To install disinformation_classification_pietervanbrakel, run this command in your terminal:

```sh
uv add disinformation_classification_pietervanbrakel
```

Or if you prefer to use `pip`:

```sh
pip install disinformation_classification_pietervanbrakel
```

## From source

The source files for disinformation_classification_pietervanbrakel can be downloaded from the [Github repo](https://github.com/PieterVanBrakel/disinformation_classification_pietervanbrakel).

You can either clone the public repository:

```sh
git clone git://github.com/PieterVanBrakel/disinformation_classification_pietervanbrakel
```

Or download the [tarball](https://github.com/PieterVanBrakel/disinformation_classification_pietervanbrakel/tarball/master):

```sh
curl -OJL https://github.com/PieterVanBrakel/disinformation_classification_pietervanbrakel/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd disinformation_classification_pietervanbrakel
uv pip install .
```
